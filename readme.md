##yandexPhoto plugin
methods:
- "showPhotoDay" - show today best photo from yandex.
- "showSlideshow" - show slideshow from best photos.

####parameters:
#####showPhotoDay
- date - = js date object.
- size -  one of  supported size of pic.

#####showSlideshow
- delay - delay slider to change
- size -  one of  supported size of pic.

#####supported sizes
- 'L',
- 'M',
- 'S',
- 'XL',
- 'XS',
- 'XXL',
- 'XXS',
- 'XXXS',
- 'orig'
